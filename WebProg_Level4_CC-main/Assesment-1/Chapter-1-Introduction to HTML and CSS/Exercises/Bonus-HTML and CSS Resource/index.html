HTML & CSS RESOURCE

HTML TAGS

<p> | Paragraph tag
- used for making paragraph content.
Example: 
<p>Hi, welcome to the Philippines!</p>
<a> | a tag
- used for adding links, supported by the href attribute.
Example: 
<a href="Home.html">Home<a>
<i> | Italics tag
- used to make text italicized.
Example: 
<p><i>Hi, welcome to the Philippines!</i></p>
<b> | Bold tag
- used for making paragraph content.
Example: 
<p><b>Hi, welcome to the Philippines!<b></p>
<ul> | Unordered List tag
- used for creating lists when order doesn't matter.
Example:
<ul>
  <li>Red</li>
  <li>Yellow</li>
  <li>Blue</li>
</ul>
<ol> | Ordered List tag
- used for creating listswhen order does matter.
Example:
<ol>
  <li>Stop</li>
  <li>Drop</li>
  <li>Roll</li>
</ol>
<li> | List item tag
- used when making points in both unordered lists and ordered lists.
Example:
<ul>
  <li>Water</li>
  <li>Food</li>
  <li>Tent</li>
  <li>Mattress</li>
  <li>Car</li>
</ul>
<dl> | Dictionary List tag
- used for creating lists with terms and descriptions.
Example:
<dl>
  <dt>Synonym</dt>
  <dd>Words with similar meanings</dd>
  <dt>Antonym</dt>
  <dd>Words with opposite meanings</dd>
</dl>
<dt> | Dictionary term tag
- used to make the terms within a dictionary list.
Example:
<dl>
  <dt>Analyze</dt>
  <dd>Examining something in detail to interpret it</dd>
  <dt>Comprehend</dt>
  <dd>Mentally grasping a concept to understand it</dd>
</dl>
<dd> | Dictionary Description tag
- used to make the decriptions within a dictionary list.
Example:
<dl>
  <dt>Skimming</dt>
  <dd>Reading quickly to get a general overview.</dd>
  <dt>Scanning</dt>
  <dd>Rapidly reading to find specific information.</dd>
</dl>

CSS PROPERTIES

text-align property
- used to align texts either to the left, right, or the center.
Example: 
text-align: center;
font-size property
- used to specify the size of the text.
Example: 
font-size: 25px;
font-family property
- used to specify the font of the text.
Example: 
font-family: 'Times New Roman', 'Times', serif;
font-weight property
- used to change the boldness of the text.
Example: 
font-weight: 700;
font-style property
- used to edit the text's font style (e.g., italics, oblique, normal).
Example: 
font-style: italic;
color property
- used to change the color of the text.
Example: 
color: black;
background-color property
- used to change the background-color of the selector.
Example: 
background-color: pink;
background-size property
- used to specify how a background fits onto the page.
Example: 
background-size: cover;
background-position property
- used to set the background's position on a page.
Example: 
background-position: top;
background-image property
- used to insert a background image within an area of a page.
Example: 
background-image: url(images/background.png);
border property
- used to add a border around a text, image, or content.
Example: 
border: 2px solid black;
border-color property
- used to change a border's color.
Example: 
border-color: green;
